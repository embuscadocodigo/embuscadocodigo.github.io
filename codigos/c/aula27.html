Alterar valor pelo ponteiro C/C++: Aula 27


Nesse vídeo vemos como mostrar em qual posição da memória o valor da variável está alocado, em seguida alteramos o valor da variável pelo ponteiro. Criamos um segundo ponteiro e repetimos tudo (o pouco) que fizemos nessas duas ultimas vídeo aulas.


Código fonte:


#include <stdio.h>
#include <conio.h>

int main () {
    int *ponteiro, valor;
    int *ponteirodois, valordois;
   
    valor =10;
    valordois = 2;
   
    ponteiro = &valor;
    *ponteiro = 100;
    ponteirodois = &valordois;
    *ponteirodois = 200;
   
    printf ("\n\n%d", valor);
    printf ("\n\n%d", ponteiro);
   
    printf ("\n\n%d", valordois);
    printf ("\n\n%d", ponteirodois);   
       
getch ();
return (0);
}
