Função dentro de outra função em C/C++. Aula 30


Como chamar uma função dentro de outra função. Assim como também vemos que não importa a ordem dentro do código fonte em que as funções são criadas, você pode chama-las quantas vezes quiser.
Não inseri exemplo de códigos "complexos" nesse vídeo mas, uma função pode conter qualquer código suportado pela Linguagem C suporta, inclusive simples palavras.


Código fonte:


#include <conio.h>
#include <stdio.h>

int outrafuncao () {
    printf (".");
    return(0);
}

int funcao () {
    printf ("funcao");
    return(0);
}

int main () {
   
    printf ("\n\n\n\nChamando uma funcao chamada: ");
    funcao();
    outrafuncao();
    printf ("\n\n\n\n\n\n");
    funcao();
   
    getch();
    return(0);
}
