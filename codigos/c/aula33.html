Instalar compilador de C/C++ no linux, aula 33


Nesse vídeo vemos como instalar um compilador de C/C++ no Linux (Elementary OS Freya, baseado no Ubuntu), compilar seu código fonte pelo terminal e, executa-lo.

Para instalar o compilador em seu Linux, no meu caso um Elementary OS que é baseado no Ubuntu 14.04 digite (em seu terminal):



sudo apt-get install gcc




Para acessar a pasta onde salvar seus arquivos .c ou .cpp utilize:



cd /pasta/destino/pastas/




Para compilar o código fonte:



gcc olamundo.cpp -o olamundo




Para executar o arquivo compilado utilize:



./olamundo



Código fonte:



#include <stdio.h>

int main () {
     
      printf ("\n\n\nOla Mundo!!!\n\n\n\n");

}
