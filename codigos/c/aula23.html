Básico em Matriz C/C++: Aula 23


Nesse vídeo utilizamos uma variável do tipo inteiro para criarmos nossa primeiro matriz. O código fonte abaixo cria um executável que "pede" para o usuário digitar os valores (do tipo inteiro) de uma matriz e logo em seguida mostra os valores inseridos e a posição dos mesmos dentro da matriz.


Código fonte:


#include <conio.h>
#include <stdio.h>

int main (void) {
    int matriz [3][3];
    int x,y;
   
    for (x=0;x<3;x++)
    for (y=0;y<3;y++) {
    printf ("\nDigite (%d,%d):", x,y);
    scanf("%d", &matriz[x][y]);
}
    for (x=0;x<3;x++)
    for (y=0;y<3;y++) {
    printf("Na posicao (%d,%d), o valor eh %d.\n", x,y, matriz[x][y]);
    }
   
getch();
return (0);
}
