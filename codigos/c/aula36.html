Limpando o buffer em C/C++. Vídeo 36


Neste exemplo pratico vemos que ao solicitarmos que o usuário digite duas variáveis do tipo char uma seguida da outra ocorre um erro.


Esse erro acontece pois ao apertar o enter o programa entende que a tecla enter é um char, mesmo não sendo.


Existem vários jeitos diferentes de se apagar a sujeirinha do buffer, muitos sites indicam que deixe dois modos diferentes, uma para windows outra para linux, mas no exemplo a seguir vemos que isso não é necessário.



Código fonte:

#include <stdio.h>

int main() {
     
      char um, dois;

      printf("Caractere: ");
      scanf("%c", &um);

      getchar();

      printf("Caractere: ");
      scanf("%c", &dois);

      printf("\n\n\nCaracteres digitados: %c %c\n\n\n", um, dois);
}
