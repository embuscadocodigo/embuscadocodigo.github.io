strlen C/C++: Tamanho de um string. Aula 21


Nesse vídeo utilizamos a strlen para verificar o tamanho de um string digitada pelo usuário, lembrando que toda string termina com um caractere nulo, mas esse caractere não é contado utilizando a strlen.

#include <conio.h>
#include <stdio.h>
#include <string.h>

getch ();
