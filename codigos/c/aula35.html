Descomplicando o SWITCH.

Vetores [parte 01].

Instalar compilador de C/C++ no linux.

Nesta videoaula vemos como utilizar o sprintf juntamente com o puts para não precisar ficar digitando um printf toda vez que precisar chamar uma variável.


Código fonte:

#include <stdio.h>

int main (void) {

      int vetor[2];
      int i, a;
      char string1[100], string2[100];

      printf("Digite um valor: ");
      scanf("%d", &vetor[0]);
      printf("Digite um valor: ");
      scanf("%d", &vetor[1]);

      sprintf(string1, "Valor = %d", vetor[0]);
      sprintf(string2, "Valor = %d", vetor[1]);

      printf("Qual posicao (1 ou 2)? ");
      scanf("%d", &a);

switch (a) {
      case 1:
            puts(string1);
      break;
      case 2:
            puts(string2);
      break;
      defaut:
            printf("Leia as opcoes (FIM).");
      break;
}
return 0;
}
