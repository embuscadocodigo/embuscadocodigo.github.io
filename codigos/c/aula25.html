Usuario digita valores da matriz (char) C/C++: Aula 25


Para entender completamente esse código fonte você precisa ter visto os dois vídeos anteriores. Nesse vídeo criamos uma matriz com valores do tipo char cujo esses valores é o usuário que digita.


Código fonte:


#include <conio.h>
#include <stdio.h>

int main () {
    char matriz [5][100];
    int variavel;
   
    for (variavel=0; variavel<5; variavel++) {
        printf ("\n\nDigite um valor: ");
        gets(matriz[variavel]);
    }
   
    printf ("\n\nA matriz que você digitou eh:\n");
   
    for (variavel=0; variavel<5; variavel++) {
        printf("%s\n", matriz[variavel]);
    }
   
getch();
return(0);
}
