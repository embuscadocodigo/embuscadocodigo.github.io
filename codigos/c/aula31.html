Chamada por valor em CC++ [função parte 2]. Aula 31


Ao invés de alterar o valor da variável digitada pelo usuário dentro da função main, neste vídeo vemos como utilizar chamada por valor em C/C++.


Código fonte:


#include <conio.h>
#include <stdio.h>

float funcao (float digitado) {
    digitado = digitado * 2;
    return digitado;
}

int main () {
    float digitado, variavel;
    printf ("Digite um valor: ");
    scanf("%f", &digitado);
   
    variavel = funcao (digitado);
   
    printf("\n\nO valor digitado eh: %f", digitado);
    printf("\n\nA varivel eh: %f", variavel);
   
getch();
return (0);
}
