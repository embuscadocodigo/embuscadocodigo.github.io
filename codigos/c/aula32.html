#define. Diretiva de Compilação em C/C++. Aula 32


Quando o compilador encontrar o nome de algum define ele substitui esse nome pela sequencia de caracteres associada ao mesmo.


Código fonte:


#include <stdio.h>
#include <conio.h>
#define nota 10.00
#define nome "Em Busca do Codigo"

int main () {
   
    printf ("O %s eh nota %.2f.", nome, nota);
   
    getch();
    return(0);
}
