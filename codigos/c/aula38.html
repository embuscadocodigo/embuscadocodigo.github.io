Usuário digita valores da struct. C/C++ 38


Nesta videoaula vemos como programar para que o usuário digite valores (números) de uma struct.


Segundo vídeo da série sobre struct, no vídeo anterior aprendemos a criar uma struct agora vemos como programar para que o usuário digite os valores das variáveis da struct. Algo extremamente importante é o parenteses a mais que encontra-se no scanf, veja no código fonte a seguir.



Código fonte:

#include <stdio.h>

struct nota {
    float nota1;
    float nota2;
};

int main() {

    struct nota notas;
    float media;

    printf("Digite a primeira nota: ");
    scanf("%f", &(notas.nota1));

    printf("Digite a segunda nota: ");
    scanf("%f", &(notas.nota2));

    media = (notas.nota1 + notas.nota2)/2;

    printf("Media = %.2f\n\n", media);

    return(0);
}
