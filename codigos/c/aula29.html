Ponteiro apontando para outro ponteiro C/C++, aula 29


Neste vídeo vemos como criar um ponteiro que aponta para o endereço de outro ponteiro, como podemos ver é bem simples o modo de se fazer isso, assim como programar é bem fácil, só ter atenção.


Código fonte:


#include <conio.h>
#include <stdio.h>

int main () {
    float valor, *ponteiro, **ponteiroparaponteiro;
   
    printf ("Digite um valor: ");
    scanf("%f", &valor);
   
    ponteiro = &valor;
    ponteiroparaponteiro = &ponteiro;
   
    printf ("Chamando pela variavel: %f.\n", valor);
    printf ("Chamando pelo ponteiro: %f.", *ponteiro);
printf ("Segundo ponteiro: %f.", **ponteiroparaponteiro);
   
    getch();
    return(0);
}
