Criar ponteiro C/C++: Aula 26


Quando você cria um ponteiro, você está falando para seu programa alocar algo na memória do seu computador para utiliza-lo mais tarde. Por exemplo: Quando você insere um contato em seu celular, o S.O. do seu celular está fazendo algo parecido. O ponteiro é só a posição da memória, por isso, em linguagem C precisamos criar uma variável do mesmo tipo para informarmos o valor que será alocado neste ponteiro, confira o vídeo novamente.


Código fonte:


#include <stdio.h>
#include <conio.h>

int main () {
    int *ponteiro, valor;
   
    valor =10;
   
    ponteiro = &valor;
   
    printf ("\n\n%d", valor);
   
getch ();
return (0);
}
