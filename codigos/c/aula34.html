Salvar variável na CPU, C/C++. Aula 34


Salvar variáveis do seu programa na CPU faz com que o mesmo fique com um desempenho melhor (mais rápido) mas, lembre-se de não usar salvar variáveis globais na CPU, pois a mesma ficaria o tempo todo ocupada por uma variável.

A sigla CPU significa Unidade Central de Processamento em inglês e, é dela que as operações do computador são processadas.

Utilizar register em C/C++ faz com que a variável  seja processada pela CPU e não pela memória, o que torna esse processo mais rápido, pois será processada diretamente pelo próprio processador.


Código fonte:



#include <stdio.h>

int main () {
      register int x;

      printf("\n\nDigite a variavel que serah salva na CPU: ");
      scanf("%d", &x);

      printf("\n\nO valor salvo na CPU eh: %d\n\n", x);

}
