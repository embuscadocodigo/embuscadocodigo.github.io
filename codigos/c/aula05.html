Programando C/C++: Aula 05, Multiplicação e Divisão

Nesse vídeo fazemos dois executáveis, um realizando uma multiplicação e outro uma divisão que retorna seu resultado e sua sobra.

Código fonte (1):
#include <stdio.h>
#include <conio.h>

int main () {
    float a,b,c;
   
    printf ("Digite um valor: ");
    scanf ("%f", &a);
    printf ("Digite um valor: ");
    scanf ("%f", &b);
    c = a * b;
    printf ("O resultado eh %.2f", c);
   
getch ();
return (0);
}


Código fonte (2):
#include <stdio.h>
#include <conio.h>

int main () {
