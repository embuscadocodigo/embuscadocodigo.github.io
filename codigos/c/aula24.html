Matriz de char C/C++: Aula 24


Neste vídeo criamos um executável com três exemplos sendo apenas o terceiro exemplo uma matriz, mas o terceiro exemplo sozinho seria apenas a continuidade do vídeo anterior. Os dois primeiros exemplos mostram duas formas de inserir o valor de uma variável do tipo char com mais de uma posição.


Código fonte:


#include <conio.h>
#include <stdio.h>

int main () {
    char matriz [10] = {'s','e','u',' ','n','o','m','e'};
    printf("Primeiro jeito: %s\n\n", matriz);
  
    char matrizdois [10] = "seu nome";
    printf("Segundo jeito: %s\n\n", matrizdois);
  
    char matriztres [3][10] = {"seu","nome","aqui"};
    int variavel;
    for (variavel=0;variavel<3;variavel++) {
    printf("%s ", matriztres[variavel]);
    }
  
getch();
return (0);
}
