Marcadores de listas com imagens em HTML5 CSS3. Vídeo 92


Neste videoaula vemos como inserir imagens nos itens de uma lista.


Como sabemos ao inserir uma lista no HTML os itens dessa lista aparecem com "círculos" por padrão, agora nós vemos como substituir esses pontinhos das listas pela imagem que você quiser.


IMPORTANTE: Esta propriedade não configura sozinha o tamanho da imagem, então, fazendo desse jeito você tem que configurar o tamanho manualmente.


A seguir você pode conferir as outras duas videoaulas onde falo de listas:


Link: Aula 06, listas não ordenadas, ordenadas e de definição
Link: Alinhar listas em bloco [inserindo produtos]. HTML5 CSS3 aula 33

A seguir vemos o código fonte desta videoaula, o mais importante está no código fonte do CSS, caso fique alguma dúvida ou tenha curiosidade assista ao vídeo.


Código fonte HTML:

<!DOCTYPE html>
<html>
     <head>
          <title>Marcadores de listas com imagens em HTML5 CSS3</title>
        <link rel="stylesheet" href="imglistas.css">
     </head>
     <body>
          <ul>
               <li>Primeiro</li>
               <li>Segundo</li>
               <li>Terceiro</li>
          </ul>
     </body>
</html>

Código fonte CSS:

ul {
     list-style-image: url("branding.png");
}
