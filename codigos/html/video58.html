Mover elementos no HTML5 CSS3, vídeo 58


Nesse vídeo vemos a primeira forma de como mover elementos no HTML5 CSS3.


Código fonte HTML:


<!DOCTYPE html>

<html>
    <head>
        <title>
            Movendo
        </title>
        <link rel="stylesheet" href="mover.css">
    </head>
    <body>
        <section class="mover">
        </section>
    </body>
<html>

Código fonte CSS:


.mover {
    height: 500px;
    width: 500px;
    background-color: green;
}
.mover:hover {
    margin-left: 200px;
    transition: 3s;
}
