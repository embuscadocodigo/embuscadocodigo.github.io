Links em Frame HTML5 CSS3, vídeo 69


Neste vídeo vemos cinco formas diferentes de se trabalhar com (direcionar) links em frames, por padrão sempre que tiver um link dentro de um frame, esse link abrirá dentro deste frame. Isso ocorre porque um frame é na verdade um documento html.


Para que isso não ocorra utilizamos o atributo target, responsável por direcionar links em HTML5, mas como isso funciona na verdade?


Por padrão um link em um frame abre dentro do próprio, e nesse tipo de caso o target poderia ser utilizado como sendo _self:



<a href="frame01.html" target="_self">Em</a>



E se eu quiser que um link dentro de um frame abra em outra aba / "janela" do navegador? Para isso podemos utilizar o valor _blank para o nosso target:



 <a href="frame02.html" target="_blank">Busca</a>



No caso de cabeçalho, o link dentro do frame atualiza a pagina toda e, para isso eu utilizo o valor _top para o atributo target:



<a href="frame03.html" target="_top">do</a>



Mas dependendo do site eu preciso que o link de um frame atualize (abra) outro frame, para isso o meu target receberá no seu valor o nome do outro frame:



 <a href="frame04.html" target="NomeDoFrame"> Código </a>



Código fonte HTML:


<ul>
      <li><a href="frame01.html" target="_self"> Em </a></li>
      <li><a href="frame02.html" target="_blank"> Busca </a></li>
      <li><a href="frame03.html" target="_top"> do </a></li>
      <li><a href="frame04.html" target="NomeDoFrame"> Código </a></li>
</ul>
