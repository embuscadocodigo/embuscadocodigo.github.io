Translate: Mover por eixos no HTML5 CSS3, vídeo 59


Em vídeos anteriores já vimos como posicionar e mover elementos, entretanto nunca vimos como move-lo por eixos, sendo apenas uma linha de código utilizando a propriedade transform.
Este é o "primeiro" vídeo utilizando a propriedade transform.


Código fonte HTML:


<!DOCTYPE html>

<html>
    <head>
        <title>transform</title>
        <link rel="stylesheet" href="translate.css">
    </head>
    <body>
        <section class="translate">
        </section>
    </body>
<html>

Código fonte CSS:


.translate {
    height: 100px;
    width: 100px;
    background-color: gray;
}
.translate:hover {
    transform: translate(20px, 20px);
    transition: 3s;
}
