Alterar tamanho por escala no HTML5 CSS3, vídeo 62


Neste vídeo alteramos o tamanho de um elemento por escala, esse elemento pode ser uma imagem. Isso mesmo e, como "sempre", utilizo hover para vermos melhor essa alteração de tamanho que utilizar escala faz.


Código fonte HTML:


<!DOCTYPE html>

<html>
    <head>
        <title>escala</title>
        <link rel="stylesheet" href="escala.css">
    </head>
    <body>
        <section class="escala">
        </section>
    </body>
</html>

Código fonte CSS:


body {
    background-color: #363636;
}
section {
    margin: 200px;
}
.escala {
    width: 50px;
    height: 50px;
    background-color: blue;
}
.escala:hover {
    transform: scale(0.5);
}
