Array em JavaScript HTML5 CSS3 aula 41


Um array em JavaScript é uma lista em que seu primeiro valor é 0 (zero).

Criando um array:


sintaxe: var umarray = ["PrimeiroValor","SegundoValor"];



Nesse vídeo criamos um array, lemos esse array, vemos o número de elementos presentes nesse array, mudamos o valor de um elemento, inserimos um novo elemento em nosso array, criamos uma string com nosso array, fazendo assim ele aparecer inteiro como uma frase, e criamos outra string que seleciona a string formada pelo meu array e separa todos os caracteres com "," (vírgula).

Chamando um array:


alert(umarray[1]) ;



No exemplo acima eu criei um alert chamando o valor que está na posição 1, se testar vai perceber que aparece o texto "SegundoValor", isso porque não podemos esquecer que um array começa a contar o que está salvo dentro dele a partir do zero, então, o valor zero é o "PrimeiroValor".

Como ver o tamanho de um array?


alert(umarray.length) ;



Neste caso aparece o número dois (2), pois temos dois valores salvos dentro desse array

Como mudar valor de um array?


umarray[1] = "Novovalor";



Como inserir um novo valor em um array?


umarray.push("inserirnovovalor");



Como chamar todos os valores contidos em um array?


var stringarray = umarray.join('');



Como criar um split em javascrip?


var umsplit = stringarray.split('');
