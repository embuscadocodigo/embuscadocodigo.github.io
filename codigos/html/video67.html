Endereço via Google Maps no site, HTML5 CSS, vídeo 67


Nesse vídeo vemos como colocar o mapa do google (Google Maps) em seu site ou blog.


Código fonte HTML:


<!DOCTYPE html>

<html>
<head>
<title>Google Mapas</title>
<link rel="stylesheet" href="mapas.css">
</head>

<body>
      <section class="mapas">
            <iframe src="https://www.google.com/maps/embed?pb=!1m18!1m12!1m3!1d3690.6000579998795!2d49.07231738556559!3d22.330960485307646!2m3!1f0!2f0!3f0!3m2!1i1024!2i768!4f13.1!3m3!1m2!1s0x94bf5da2d4399001%3A0x3d4e41719c53ef34!2sPrefeitura+Municipal+de+Bauru!5e0!3m2!1spt-BR!2sbr!4v1457038184956" width="500" height="150" frameborder="0" style="border:0" allowfullscreen></iframe>
      </section>
</body>
</html>


Código fonte CSS:


.mapas {
      text-align: center;
      width: 100%;
}
