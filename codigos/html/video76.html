Link em iframe no HTML5 e CSS3, vídeo 76


Neste vídeo vemos três tipos de direcionamento (redirecionar) links que estão dentro de iframes.
O primeiro é uma tag a simples, e vemos que em iframes o padrão é que links sejam abertos dentro do iframe.



<a href="oqueprecisar.html">Clique aqui.</a>



O segundo exemplo é quando precisamos abrir o link de um iframe dentro de outro iframe, para isso utilizamos um simples target com o nome do outro iframe.



<a href="link.html" target="nome_do_outro_iframe">Clique aqui.</a>



O terceiro exemplo, que é o exemplo que usarei em meus projetos daqui para frente é quando precisamos abrir o link dentro do iframe em um documento html para todo o site, abrir o iframe em outra pagina e na mesma aba, atualizando a pagina atual. Veja o exemplo a seguir:



<a href="principal.html" target="_top">Clique aqui.</a>



Código fonte HTML (principallinks.html):


<!DOCTYPE html>

<html>
      <head>
            <title>01</title>
      </head>

      <body bgcolor="black">
            <iframe name="um" src="iframelinks.html" width="100%" height="50px"
            frameborder='0' scrolling="no">
            </iframe>
            <iframe name="dois" src="iframelinks2.html" width="20%" height="750px"
            frameborder='0' scrolling="no">
            </iframe>
      </body>
</html>

Código fonte HTML (novo.html):


<!DOCTYPE html>

<html>
      <head>
            <title>02</title>
      </head>

      <body bgcolor="white">
            <iframe name="um" src="iframelinks.html" width="100%" height="50px"
            frameborder='0' scrolling="no">
            </iframe>
            <iframe name="dois" src="iframelinks2.html" width="20%" height="750px"
            frameborder='0' scrolling="no">
            </iframe>
      </body>
</html>

Código fonte HTML (iframelinks.html):

<html>
      <head>
      </head>

      <body bgcolor="gray">
            <a href="novo.html" target="_top">Clique aqui.</a>
