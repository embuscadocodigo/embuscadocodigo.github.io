Álbum modal em HTML5 CSS3. Vídeo 99


Na videoaula anterior nós vimos como funciona uma imagem modal, o que é um modal e como fazer para que uma imagem abra em um modal.


Agora nós criamos um álbum de fotos modal, isso mesmo, um álbum de imagens que abra dentro de um modal.


Alterei minimamente o código fonte anterior, então, antes de darmos continuidade nesta "aula"crie um documento HTML e outro CSS, lembrando, estes dois documentos podem ter qualquer nome, mas não se esqueça de linkar seu documento CSS no documento HTML.


Código fonte HTML (inicial):

<!DOCTYPE html>

<html>
     <head>
          <meta http-equiv="Content-type" content="text/html; charset=UTF-8">
          <title>Imagem modal</title>
          <link rel="stylesheet" href="modal.css">
     </head>
     <body>
          <a href="#Bookmark01">
               <figure>
                    <img src="img01.png" width="12%"></img>
                    <figcaption>imagem</figcaption>
               </figure>
          </a>
          <div id="Bookmark01" class="dialogo">
               <div class="imgfake">
                    <a class="fechar" title="Fechar" href="#qualquernome">X</a>
                    <img class="imagemmodal" src="img01.png"></img>
               </div>
          </div>
     </body>
</html>

Código fonte CSS (inicial):

.dialogo {
     position: fixed;
     top: 0;
     right: 0;
     left: 0;
     bottom: 0;
     background: rgba(255,193,193,0.5);
     z-index: 11111111;
     opacity: 0;
     pointer-events: none;
     overflow: scroll;
}

.dialogo:target {
     opacity: 1;
     pointer-events: auto;
}

.imgfake {
     position: relative;
     margin-left: 25%;
     margin-top: 5%;
}

.imagemmodal {
     display: block;
     float: center;
     width: 50%;
}
.fechar {
     color: #ffffff;
     position: absolute;
     text-align: left;
     width: 24px;
     text-decoration: none;
     font-weight: bold;
}

Antes de assistir ao vídeo tenha em mente que para fazer seu álbum modal você tem que saber o número de imagens que terão em seu álbum (isso utilizando somente "HTML5 e CSS3").


Mas o que nós vemos no vídeo?


A primeira coisa que fazemos é inserir todas as imagens no HTML, essa parte eu dei Ctrl+C / Ctrl+V mesmo pois o posicionamento das imagens no seu site depende do seu layout (depende de você), nesse tutorial o foco é fazer um álbum modal.
