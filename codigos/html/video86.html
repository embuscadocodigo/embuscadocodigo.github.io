Letras adaptativas em HTML5 CSS3. Vídeo 86


Essas letras se adaptam ao dispositivo ou navegador que o usuário está utilizando.


Ao invés de usar px para configurar o tamanho da fonte, utilize em. Por padrão 1em equivale ao padrão de letras (normais) do dispositivo ou navegador (celular, tablet ou pc). Quando for um desktop o tamanho padrão das letras é 16px, então, no pc 1em é igual a 16px. Mas isso muda quando o user utiliza dispositivos mobiles, assim como tablets, smartphones e celulares mais antigos.


Os links a seguir podem te auxiliar na parte teórica, caso a parte pratica não seja o suficiente.

Link externo (abre em outra aba): Dicas & truques CSS
Link externo (abre em outra aba): CSS Fonts

Utilize o código fonte a seguir para ver como as letras se comportam em diferetes dispositivos, e, utilize as que mais lhe convém.


Código fonte HTML:


<body>
      <p class="um">Olá, Mundo!!!</p>
      <p class="dois">Olá, Mundo!!!</p>
      <p class="tres">Olá, Mundo!!!</p>
      <p class="quatro">Olá, Mundo!!!</p>
      <p class="cinco">Olá, Mundo!!!</p>
</body>


Código fonte CSS:


.um {
      font-size: 1em; /*padrão de navegadores desktop*/
}
.dois {
      font-size: 2em;
}
.tres {
      font-size: 3em; /*Tamanho do H1 no desktop*/
}
.quatro {
      font-size: 4em;
}
.cinco {
      font-size: 5em;
}
