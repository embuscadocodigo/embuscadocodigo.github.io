Velocidade da animação no HTML5 CSS3: vídeo 57


Nesse vídeo vemos como configurar a velocidade de uma animação utilizando CSS3, quatro modos diferentes onde linear configura uma velocidade constante, ease configura uma velocidade onde a mesma diminui gradativamente, ease-in configura a velocidade da animação para aumentar gradativamente e, ease-in-out aumenta e logo depois reduz a velocidade da animação.


Código fonte HTML:


<html>
    <head>
        <link rel="stylesheet" href="cvideo57.css">
    </head>
    <body>
        <section class="tempo">
        </section>
    </body>
</html>

Código fonte CSS:


.tempo {
    width: 20%;
    height: 50px;
    background-color: blue;
}
.tempo:hover {
    background-color: red;
    width: 90%;
    /* transition: 5s linear; velocidade constante*/
    /* transition: 5s ease; reduz */
    /* transition: 5s ease-in; aumenta */
    transition: 5s ease-in-out; /* aumenta depois diminui */
