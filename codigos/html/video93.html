Valores de list style type em HTML5 CSS3. Vídeo 93


Aqui vemos 21 exemplos de marcadores de listas em HTML diferentes, alguns como o decimal insere os valores da lista a partir do número 1, 2 , 3, e assim por diante.


Mas também como configurar nossa lista em HTML em ordem alfabética, por números romanos, em latin, ordenar por letras orientas e até "hebrew", e mais (assista ao vídeo).


Esta mini série especial sobre listas em HTML5 CSS3 tem uma única fonte, esta fonte é o Maujor (link externo da fonte logo após o vídeo desta videoaula). Se você chegou até aqui você deve ser informado que este curso atualmente chamado de HTML5 CSS3 chegará ao seu fim na videoaula número 100.


Não chega a ser um fim, pois ainda existem muitos conceitos que considero importante e que ainda não estudei, mas é necessário seguir em frente e aprender novas linguagens, muito além de uma linguagem de marcação. Mas não se engane, desenvolvedor web é programador sim.


A seguir você o código fonte HTML, depois o CSS e logo em seguida como fica o código fonte.



Código fonte HTML:

<!DOCTYPE html>

<html>

     <head>
          <title>list-style-type COMPLETO</title>
          <meta http-equiv="Content-type" content="text/html; charset=UTF-8">
          <link rel="stylesheet" href="liststyletype.css">
     </head>

     <body>
          <ul >
               <li class="none">Olá</li>
               <li class="disc">Mundo!!!</li>
               <li class="circle">Olá</li>
               <li class="square">Mundo!!!</li>
               <li class="decimal">Olá</li>
               <li class=" decimal-leading-zero">Mundo!!!</li>
               <li class="lower-roman">Olá</li>
               <li class="upper-roman">Mundo!!!</li>
               <li class="lower-alpha">Olá</li>
               <li class="upper-alpha">Mundo!!!</li>
               <li class="lower-greek">Olá</li>
               <li class="upper-latin">Mundo!!!</li>
               <li class="hebrew">Olá</li>
               <li class="armenian">Mundo!!!</li>
               <li class="georgian">Olá</li>
               <li class="cjk-ideographic">Mundo!!!</li>
               <li class="hiragana">Olá</li>
               <li class="katakana">Mundo!!!</li>
               <li class="hiragana-iroha">Olá</li>
               <li class="katakana-iroha">Mundo!!!</li>
               <li class="lower-latin">Olá</li>
               <li>Mundo!!!</li>
          </ul>
     </body>

</html>
