Carrinho de compra [parte 01]. HTML5 CSS3 aula 35


Nesse vídeo vemos o inicio de como inserir um carinho de compras, no final desse vídeo o carrinho não está funcionando ainda, pois esse tutorial está seguindo uma ordem e primeiramente inserimos a base do nosso site, antes de vermos conceitos mais avançados.


Código fonte HTML



<nav class="cabecalho">
<div class="cabeloja">
      <ul>
<!-- .... -->
</div>
            <div class="compras">
                  <a href="carrinho.html">
                        <ol> <!--sub lista da primeira lista-->
                              <li>Minhas compras</li>
                              <li><img class="carrinho" src="css/img/01.png"></li>
                        </ol>
                  </a>
            </div>
      </ul>

Código fonte CSS:



.compras img {
      height: 30px;
      width: 30px;
      line-height: 10px;
}
.compras {
      font-size: 12px;
      padding-right: 35px;
      text-align: right;
      width: 150px;
      height: 25px;
      position: relative;
      float: right;
      bottom: 14px;
      clear: right;
}
.compras ol li {
      display: inline;
      text-align: center;
      list-style-type: none;
}
.carrinho {
      position: absolute;
      /*right: Npx; depende da imagem ela pode estar tocando no seu texto*/
      bottom: 5px;
}
.compras a {
      text-decoraton: none;
}
.cabeloja {
      left: 150px;
}
