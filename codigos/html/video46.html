Pseudo-classes, Aula 46 HTML5 CSS3


Nesse vídeo vemos um exemplo de como utilizar pseudo classe em CSS3 quando queremos configurar apenas o primeiro item de uma lista ou o ultimo.


Código fonte HTML:

<!--...-->
<section class="paragrafos">
      <ul>
            <li>No CSS3</li>
            <li>li:first-child {...</li>
            <li>Configura o primeiro item</li>
            <li>li:last-child {..</li>
            <li>configura o ultimo item</li>
            <li>Sem precisar criar uma classe</li>
      </ul>
</section>
<!--...-->

Código fonte CSS:

.paragrafos li:first-child {
    color: blue;
}
.paragrafos li:last-child {
    color: red;
}
