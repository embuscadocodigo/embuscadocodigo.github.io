HTML5 CSS3. Sombra em texto vídeo 51


Nesse vídeo vemos como inserir sombras em textos, sombras simples, espelhadas, mais de uma sombra no mesmo texto, isso via CSS. Facebook: Em Busca do Código.

Código fonte HTML:


<!DOCTYPE html>

<html>
<head>
<link rel="stylesheet" href="sombra.css">
</head>
<body>
<p>Olá Mundo !!!</p>
</body>
</html>

Código fonte CSS:

p {
    text-shadow: 5px 5px 2px gray,
    -5px -5px 2px red;
}
