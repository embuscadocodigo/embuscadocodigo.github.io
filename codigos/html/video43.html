Criar banner rotativo. HTML5 CSS3 aula 43


Nesse vídeo de menos de 7 minutos criamos um banner rotativo feito de imagens e gifs, onde ele fica alterando sozinho (as imagens), utilizado bastante em cabeçalhos.

Utilizamos conceitos de HTML CSS e JavaScript para cria-lo entretanto, todos navegadores mais "modernos" já suportam JavaScript normalmente, por isso "não teremos" problemas com usuários.

Código fonte HTML

<section>
       <figure class="gif">
             <img src="css/img/004.gif">
       </figure>
 </section>

Código fonte JavaScript

var imagens = ["img01.png", "img02.png", "img03.png", "img04.png"];
var imagematual = 0;

function trocaimagem() {
 imagematual = (imagematual + 1) % 4;
document.querySelector('.gif img'). src = imagens[imagematual];
}
setInterval(trocaimagem, 1500);

Código fonte CSS (indiferente)

.gif {
       text-align: center;
       padding: 15px;
}

Vale a pena enfatizar que neste caso todas as imagens e o gif tem as mesmas dimensões.
