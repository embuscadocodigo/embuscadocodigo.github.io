Operadores de comparação em JavaScript (if e else). HTML5 CSS3 aula 42


Nesse vídeo vemos operadores de comparação em JavaScript, assim como comparar se a variável é igual, diferente, maior, menor, maior ou igual e menor ou igual.


<script>
numero = 6

if (numero == 6) {
 alert (igual)
}else{
alert(diferente)
}
</script>



Mas se quiser que verifique que é diferente você utiliza !=, se quiser comparar que é maior que você utiliza >, menor que <. Maior ou igual que você utiliza >= e, menor ou igual você utiliza <=.
