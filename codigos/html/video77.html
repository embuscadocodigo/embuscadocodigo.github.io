Pagina 404 perfeita. HTML5 e CSS3. Vídeo 77


Quase um ano depois de criar o site do Em Busca do Código tomei coragem e corrigi a tão famigerada pagina 404, mais conhecida como erro 404.

Não existe código fonte para esta videoaula, é mais uma questão de entender seu código fonte e, escolhendo fazer deste jeito você deve estar preparado para sempre editar o CSS da pagina 404, sempre quando o padrão do seu site for alterado.

É mais uma questão de dar importância e valor a uma página de erro, não apenas a uma página mas sim ao seu usuário. Quanto vale para você que o usuário do seu site não se sinta perdido, mesmo quando ele cair em um erro tão comum?

Utilizando a tag style simplesmente adicionei o código fonte dos documentos CSS para dentro do head do meu documento html.

Quando a pagina 404 era acionada de dentro de uma pasta do meu site, a pagina do erro 404 não conseguia encontrar o caminho do CSS, deixando minha pagina 404 sem CSS.

Isso também ocorria com imagens, e no caso das imagens, ao invés de eu chamar as imagens internamente, linkei as mesmas a partir do link externo dela, isso mesmo, toda imagem que está em seu site tem um link publico (até segunda ordem).

E por ultimo, passei as configurações das imagens para dentro da tag style que acabei de criar.

No caso do meu layout tive que fazer outra pequena configuração CSS dentro da tag style, mas muito provavelmente no seu caso, haverá uma ou outra peculiaridade diferente da minha, mas o conceito é o mesmo, passar as configurações que estão dentro dos documentos CSS para dentro do head do seu sua pagina do erra 404.
