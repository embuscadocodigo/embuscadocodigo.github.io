Degradê (angulos) HTML5 CSS3, vídeo 54


Quatro modos diferentes de configurar o ângulo do seu degradê feito em HTML5 CSS3.


Código fonte HTML:


<!DOCTYPE html>
<html>
    <head>
        <link rel="stylesheet" href="angulo.css">
    </head>
    <body>
        <section class="anguloum">
        </section>
        <section class="angulodois">
        </section>
        <section class="angulotres">
        </section>
        <section class="anguloquatro">
        </section>
    </body>
</html>

Código fonte CSS:


.anguloum, .angulodois, .angulotres, .anguloquatro {
    height: 100px;
    width: 700px;
    margin: 10px;
}
.anguloum {
    background: linear-gradient(100deg, red 0%, blue 100%);
}
.angulodois {
    background: linear-gradient(50grad, red 0%, blue 100%);
}
.angulotres {
    background: linear-gradient(1rad, red 0%, blue 100%);
}
.anguloquatro {
    background: linear-gradient(0.25turn, red 0%, blue 100%);
}
