Translate: Distorção no HTML5 CSS3, vídeo 61


Nesse vídeo distorcemos um elemento (podendo ser uma imagem) utilizando HTML5 e CSS3, mostrado três modos diferentes de distorcer algo em seu site ou blog. Utilizado o efeito hover para que você possa visualizar melhor a distorção em tempo real mas, o uso do hover não é obrigatório.


Código fonte HTML:


<!DOCTYPE html>

<html>
    <head>
        <title>distorcer</title>
        <link rel="stylesheet" href="distorcer.css">
    </head>
    <body>
        <section class="distorcer">
        </section>
    </body>
<html>

Código fonte CSS:


.distorcer {
    height: 100px;
    width: 100px;
    background-color: red;
    margin: 100px;
}
.distorcer:hover {
    transform: skew(10grad, 10grad);
}
