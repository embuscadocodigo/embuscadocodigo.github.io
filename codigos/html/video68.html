Frame - Um cabeçalho (documento) para todo site, vídeo 68


Pelo o que eu entendi utilizar frame em seu site é como usar uma mascara em seus documentos HTML, quando você utiliza frames, o que o seu usuário vê na verdade é a união de vários documentos html. Isso mesmo, utilizando frames você estará criando um documento que poderá estar em todas as páginas do seu site (assim como o cabeçalho).


Assistindo ao vídeo você aprenderá que para se utilizar frames você precisa de duas tags, a tag pai chamada frameset e a tag filho chamada frame. No vídeo divido a página inteira em três frames e cito dois exemplos, dividir a página html (site) na vertical e na horizontal.


Código fonte HTML (frames dividem na horizontal):


<!DOCTYPE html>

<html>
      <head>
            <title>Frame 01 doc 2</title>
      </head>
      <frameset rows="20%,*,10%" frameborder="0">
            <frame name="cabecalho" src="01.html">
            <frame name="body" src="02.html">
            <frame name="rodape" src="03.html">
      </frameset>
</html>

Código fonte HTML (frames dividem na vertical):


<!DOCTYPE html>

<html>
      <head>
            <title>Frame 01 doc 1</title>
      </head>
      <frameset cols="10%,*,20%" frameborder="0">
            <frame name="cabecalho" src="01.html">
            <frame name="body" src="02.html">
            <frame name="rodape" src="03.html">
      </frameset>
</html>

Código fonte HTML (frames):


<!DOCTYPE html>

<html>
<head></head>
<body bgcolor="green"></body>
</html>
