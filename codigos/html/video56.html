Criar movimento no HTML5 CSS3, vídeo 56


Nesse vídeo criamos movimento quando o usuário passar o mouse por cima do elemento em questão. Já vimos isso antes com o "hover" só que agora o elemento demora um tempo para chegar ao resultado final, gerando um movimento em que todos percebemos.


Código fonte HTML:


.paragrafoscodigos:hover {
    font-size: 15px;
    transition: 5s;
    border: 0px;
    border-radius: 10px;
    width: 100%;
    margin: 0px;
    width: center;
}
.paragrafos:hover {
    font-size: 15px;
    transition: 5s;
    border: 0px;
    border-radius: 10px;
    width: 100%;
    margin: 0px;
    width: center;
}
.fonte:hover {
    width: 75%;
    border-radius: 5px;
    border: 0px;
    transition: 5s;
}
