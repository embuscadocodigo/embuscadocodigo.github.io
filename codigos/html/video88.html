Áudio em HTML5 CSS3. Vídeo 88


Neste videoaula criamos um player que toca um áudio em HTML5. Para isso utilizamos a tag áudio "<audio>".


O código a seguir tem tudo o que você precisa para inserir um áudio em seu site ou blog. Uma configuração que somente comentei nesse código fonte foi a preload, essa configuração informa ao navegador para o áudio/musica começar a carregar assim que a pagina foi carregada, porém, ele não se inicia automaticamente.


A função preload deve substituir a função autoplay, função esta que faz o áudio se iniciar automaticamente quando a página for carregada


Código fonte HTML:

<!DOCTYPE html>

<html>
      <head>
            <title>audio em HTML5 CSS3</title>
      </head>

      <body>
            <audio controls="controls" autoplay="autoplay" loop="loop">
            <!--ou você usa autoplay, ou você usa preload-->
            <source src="qualqueraudio.mp3" type="audio/mp3">
            se o navegador for antigo, ele não aceita a tag audio
            <!--em navegadores antigos a mensagem acima aparecerá.-->
            </audio>
      </body>
</html>
