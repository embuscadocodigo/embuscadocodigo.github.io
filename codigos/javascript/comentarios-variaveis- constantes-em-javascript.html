Comentários em JavaScript

Para fazer uma comentário de uma linha em JS utilizamos duas barras "//".

Comentários com uma linha em JavaScript

// comentários em uma linha

Quando o comentário ter maisde uma linha utilizamos [barra]+[asterisco] "/*" para abrir o inicio do comentário e [asterisco]+[barra] "*/" para fechar.

Comentários com mais de uma linha em JavaScript

/*
comentários em
mais de
uma linha
*/

Variáveis em JavaScript

Logo em seguindo conferimos dois tipos de variáveis em JS, var e let. var cria uma variável global e let uma variável local, veremos um exemplo disso na pratica na próxima videoaula onde conferimos funções.

Variáveis em JavaScript:

/*
var nomedaVariavel = 5;
console.log(nomedaVariavel);*/
/*
let nomedaVal = 7;
console.log(nomedaVal);*/

Esses dois códigos fontes estão comentados e existe um motivo para isso, crie algumas variáveis a partir do Scratchpad do seu navegador e entenda o porque eu entreguei esse código fonte comentado.

E por ultimo e não menos importante vemos uma constante, o mais importante e o diferencial de uma constante é que não se pode alterar o valor de uma constante depois de criada.

Constantes em JavaScript:

const somenteLeitura = 'qualquer coisa';
console.log(somenteLeitura);
