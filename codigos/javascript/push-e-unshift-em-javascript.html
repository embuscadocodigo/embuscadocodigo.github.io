Adicionar elemento no inicio ou fim de um array em JavaScripit. Dica 09

Nesta dica utilizamos os métodos push e unshift para adicionar elementos no inicio ou fim de um array em JavaScript.

Objetos e métodos, e seguindo esse padrão nós utilizaremos os métodos push e unshift.

Método push

Para utilizar o método push basta chama-lo depois do nome do array. Segue o exemplo a seguir.

array1.push("novo valor");

Antes de escrever essa linha precisamos de um array.

var array1 = new Array(123,654,"oi");
document.write("<h1>Array original: "+array1+"</h1>");

Agora sim podemos utilizar o método push.

array1.push("novo valor");
document.write("<h1>Adicionando no fim: "+array1+"</h1>");

Não podendo esquecer de "criar" um documento HTML simples.

Publicidade

Tendo como base o array que acabamos de utilizar agora utilizaremos o método unshift para adicionar um elemento no índice 0 (zero) de um array.

Objetos e métodos.

array1.unshift("1º elemento");

Agora segue o código fonte JavaScript completo.

Código fonte JavaScript

document.write("<h1></br>NO FIM (push)</h1>");

var array1 = new Array(123,654,"oi");
document.write("<h1>Array original: "+array1+"</h1>");
array1.push("novo valor");
document.write("<h1>Adicionando no fim: "+array1+"</h1>");

document.write("<h1></br>NO INICIO (UNSHIFT)</h1>");

array1.unshift("1º elemento");
document.write("<h1>Adicionando no inicio: "+array1+"</h1>");
