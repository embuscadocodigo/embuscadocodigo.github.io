Array em JavaScript. Vídeo 06

A primeira informação que vemos no vídeo é a tag script. Para utilizar script (JavaScript) em um documento/arquivo HTML nós usamos a tag script

<script></script>

Como já sabemos, o JavaScript trabalha com objetos e, para utilizar um objeto nós precisamos de um método.

objeto.metodo();

Para vermos nosso array (pelo navegador) utilizaremos o objeto document com o método write.

document.write();

Para escrever números int ou float basta escrever o número entre os parênteses.

document.write(456654);

E se for uma string escrevemos ela entre aspas duplas.

document.write(“Olá Mundo!!!”);

Publicidade

Caso tenha interesse em ler mais sobre document.write no final deste artigo terá um link muito bom da Imaster, mas vamos dando continuidade em array em JavaScript.

Antes de criarmos nosso primeiro arquivo “.js” confira este código fonte até aqui.

<!DOCTYPE html>

<html>
    <head>
        <title>TAG script</title>
        <meta charset="UTF-8">
    </head>
    <body>
        <script>
            document.write(“Olá Mundo!!!”);
        </script>
    </body>
</html>

No decorrer deste curso eu sempre vou utilizar JavaScript a partir de um arquivo “.js”. E para isso eu crio um arquivo com um nome qualquer com a extensão “.js” e "linko" ele dentro da tag script

<script src="nomedodocumento.js"></script>

Array em JavaScript

A partir de agora veremos três modos de se criar arrays em JavaScript.

Primeiro modo:

var primeiroArray = new Array("textos",456,45.5);
document.write(primeiroArray);

Segundo modo:

var segundoArray = Array("elemento",primeiroArray,45);
document.write("<h1>"+segundoArray +"</h1>");

Terceiro modo:

var terceiroArray = [456,"string",2];

document.write("<h1>Terceiro Array: "+terceiroArray +"</h1>");
document.write("<h1>Indice [0]: "+terceiroArray[0] +"</h1>");
document.write("<h1>Indice [1]: "+terceiroArray[1] +"</h1>");
document.write("<h1>Indice [2]: "+terceiroArray[2] +"</h1>");
