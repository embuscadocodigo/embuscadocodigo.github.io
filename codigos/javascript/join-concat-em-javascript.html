join e concat em JavaScript. Dica 08

Se você assistiu ao vídeo anterior reparou que, em um determinado momento apareceu concat.

Seguindo o nosso curso de dicas em JavaScript em frente agora vemos dois métodos muito importantes e utilizados em JavaScript. Estamos falando dos métodos join e concat.

Mas antes de falarmos sobre concatenar arrays em JavaScript vamos ver o método join.

Quando utilizamos document.write para mostrar os valores de um array em nosso documento HTML utilizando JavaScript vemos que os elementos do array aparecem separados por , (vírgula). Como retirar as vírgulas entre os elementos do array? Como inserir outro caractere ou até mesmo strings entre os elementos de um array?

Usando join

Para substituir as vírgulas que aparecem entre os elementos de um array nós utilizamos o método join, mas antes e darmos continuidade precisamos de um código fonte inicial, eu separo o HTML do JavaScript então, a seguir você vê o código fonte inicial deste vídeo.

HTML

<!DOCTYPE html>

<html>
    <head>
        <title>Array</title>
        <meta charset="UTF-8">
    </head>
    <body>
        <h1>Array parte 3</h1>
        <script src="array03.js"></script>
    </body>
</html>

JavaScript

var array1 = new Array(0,1,2);
document.write("<h1>Primeiro array: "+ array1 + "</h1>");

Seguindo o código fonte acima vemos que existe um array qualquer e, para usarmos o método join precisamos criar uma variável.

var subvir

Logo após criarmos a variável precisamos iguala-la ao array que queremos substituir as vírgulas.

var subvir = array1

Adicionamos o método join.

var subvir = array1.join();

E entre os parênteses e aspas duplas inserir a string que será o valor do novo elemento.

var subvir = array1.join("novo valor");

Pode ser um int ou float?

Isso você pode testar sozinho.

Agora nós precisamos apenas "chamar" a variável criada.

document.write("<h1>join: "+ subvir + "</h1>");

Concatenar arrays em JavaScript

Para concatenar arrays precisamos de no mínimo dois arrays.

var array2 = new Array("oi",456,987);
document.write("<h1>Segundo array: "+ array2 + "</h1>");

Agora voltaremos um pouco ao pensamento lógico. Preciso concatenar um array.

array1

Publicidade

Em JavaScript o primeiro array informado na hora de concatenar tem que ser o primeiro declarado. Esse array estará recebendo os valores deles mesmo e mais outro array.

array1 = array1

Antes de informar qual o segundo array precisamos utilizar o método concat.

array1 = array1.concat();

E entre os parênteses informamos o nome do segundo array.

array1 = array1.concat(array2);

Neste inicio deste curso eu estou utilizando document.write para "imprimir" de uma maneira visível e eficaz o código JS no HTML, agora, sobre criar alertas, eu não gostei de utiliza-los como exemplo, ninguém é obrigado a aprender alert(); sem fazer algo útil com ele e, document.write é bem mais útil que alert.

Código fonte JavaScript

document.write("<h1>Primeiro array (concat): "+ array1 + "</h1>");
