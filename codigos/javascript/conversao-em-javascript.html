Conversão de tipos em JavaScript. Vídeo 05

A partir de agora você confere conversão de tipos em JavaScript.

Em JavaScript você não precisa falar (especificar) o tipo de variável que está criando, isso porque é uma linguagem dinamicamente tipada, que significa que, o tipo de variável (int, string, float [...]) são convertidos automaticamente conforme a execução do script.

Na aula número 04 deste curso (vídeo anterior) um pouco antes do código fonte final de JavaScript, vídeo este que tem menos de 7 minutos, duas variáveis do tipo inteiras são convertidas para string, no decorrer desta aula vamos ver um exemplo desse.

Observação: Faça estes exemplos somente com o Scratchpad, escolhi usar somente o Scratchpad nesta aula por tornar a mesma bem mais rápida. E no final de cada código fonte aqui apresentado, sempre clicar em display ou visualizar para seguir os exemplos, se quiser crie os seus depois de testar esses.

Quando queremos criar (declarar) uma variável do tipo string, no valor desta variável nós utilizamos aspas duplas, segue exemplo.

var x = "string qualquer";

E para "chama-la" (no Scratchpad) nós apenas colocamos o o nome da mesma na linha de baixo e clicamos em visualizar ou display.

E se eu quiser unir (concatenar) um número ao final desta string?

Se eu quiser unir um inteiro a essa string eu utilizo "+" para concatenar, vamos lá?

var x = "string"+ 12;
x

Esse número virou uma string, juntamente com a string existente. Faça todos os exemplos aqui descritos em seu notebook ou pc.

Posso fazer ao contrario também.

var x = 12+ " string";
x

Minha variável x que era um inteiro se tornou uma string.

E se eu alterar o valor de uma variável?

var x = 12;
x = "doze";
x

Como JavaScript é dinamicamente tipada ela muda o tipo de variável quando necessário, conforme a execução do script.

E se eu receber um número como se fosse uma string, posso fazer isso?

Sim, podemos fazer isso.

var x = "12";
x

E se eu quiser somar um número que foi declarado como uma string?

var x = "12" + 2;
x

A resposta disto será 122.

Nesse caso os dois números foram convertidos em string.

Então cuidado, isso foi o que aconteceu no vídeo anterior, só que este é outro exemplo.

Subtrair funciona.

var x = "12" - 2;
x

Multiplicar também.

var x = "12" * 2;
x

Mas também posso converter essa string (número declarado entre aspas) em int e float, e para isso utilizamos parseInt ou parseFloat

x = parseInt("12");

x = parseFloat("12.5") ;
