Funções em JavaScript. Vídeo 04 do curso de JS

O primeiro exemplo visto no vídeo é somar utilizando uma função em JavaScript, só que essa função não faz a soma sozinha e para isso criamos três variáveis.

function soma (x , y) {
     return "”+ (x + y);
};
     var x = 4;
     var y = 5;
console.log(soma);

Ao fazer este exemplo em seu navegador verá que o resultado que aparece no console é:

function soma()

Isso porque não fizemos a soma em s.

Para fazer uma soma em JavaScript (utilizando função) criamos uma outra variável e, para deixar claro o que ela faz damos a ela o nome de "resultado" e o valor dessa variável será a função "soma".

var resultado = soma (x,y);

Publicidade

Mas ainda não acabou, em nosso primeiro código fonte desta aula nós pedimos ao console para que seja mostrada diretamente a nossa função, sendo que nós queremos o resultado da função. Então nós chamamos a nossa variável resultado.

console.log(resultado);

Não vou facilitar para ninguém não. Você que insira a nossa variável "resultado" e o console.log no lugar certo.

Olá Mundo!!!

Se você quiser pode clicar em play no vídeo, pois o que está escrito aqui eu falo lá. E logo depois de realizar esta soma eu copio e colo o olá mundo que está no código fonte do primeiro vídeo deste curso.

Mas eu não utilizo este código fonte, mas falo uma coisa que eu acho importante também escrever. A maioria só lê, não aperta o play.

Mas tem outro motivo de não utilizar o código fonte e mesmo assim o inserir no meio do vídeo. Explicar o passo a passo da construção de uma função.

function nomeDaFuncao () {
};
console.log(nomeDaFuncao);

Este é o primeiro escopo do que preciso, ao chamar uma função no console aparece a palavra function o nome da função e abre e fecha parênteses.

E resumindo essa parte do vídeo mostro que podemos escrever algo entre as aspas do return de uma função, mas porque?

function nomeDaFuncao (param) {
  return "Escrever "+param;
};
console.log(nomeDaFuncao(""));

Porque a nossa função soma pode ficar ainda mais bonita se junto com o resultado estar escrito algo antes. Segue o código fonte da nossa primeira soma em JavaScript.

function soma (x , y) {
  return "Resultado "+ (x + y);
};
    var x = 4;
    var y = 5;
    var resultado = soma (x,y);
console.log(resultado);

Agora eu te pergunto, vale a pena clicar em play?
