Length. Array em JavaScript parte 2. Dica 07

Para realizar esta dica você pode utilizar o código fonte HTML da dica anterior e, utilizar apenas um dos arrays criados como exemplo.

Na dica anterior nós criamos Arrays e, nesta dica iremos contar o número de elementos de um array. Contar quantos caracteres um determinado array tem (quando este elemento é uma string), ver o ultimo elemento de um array e, diminuir ou aumentar o tamanho de um array.

Sempre utilizando a propriedade length.

Para contar quantos elementos tem um array apenas inserimos ".length" depois que chamamos o nome do array.

document.write("<h1>Array: "+Array02.length +"</h1>");

Quando um determinado elemento for uma string eu posso contar o número de caracteres que ele tem, e para isso eu só preciso dizer qual o índice do elemento ao qual eu quero contar o número de caracteres.

document.write("<h1>Array: "+Array02[0].length +"</h1>");

Para conseguir ver o ultimo elemento de um array eu preciso entender lógica, a primeira coisa que devemos fazer é criar uma variável com o nome qualquer.

var verUltimo

Como eu quero que, ao chamar essa variável apareça o ultimo elemento do meu array, eu preciso informar o array que estou utilizando.

var verUltimo = Array2[];

Agora vamos fingir que não sabemos o número de elementos desse array, mas sabemos que Array2.length (Array2 é o nome do meu array) mostra o número de elementos.

var verUltimo = Array[Array2.length];

Publicidade

Só que o primeiro elemento tem índice 0 (zero), e não 1 (um), então, do jeito que está, esta variável não teria um valor, caso tenha ficado duvida nessa parte assista ao vídeo, no vídeo eu mostro essa diferença entre o número de elementos e maior índice

Para resolver isso temos que subtrair do tamanho do nosso Array, tendo como resultado o número do último índice.

var verUltimo = Array[Array2.length - 1];

A partir daí podemos chamar essa variável, que por algum acaso sempre irá retornar o valor do ultimo elemento do array.

Podemos também excluir/diminuir o tamanho de um array.

Array02.length = 2;
document.write("<h1>Array: "+Array02 +"</h1>");
document.write("<h1>Tamanho: "+Array02.length +"</h1>");

E se o valor do length fosse maior não menor que o valor do array, o tamanho do array aumentaria. Isso acontece porque o número de elementos de um array se adapta ao valor do length.

Array02.length = 5;
document.write("<h1>Array: "+Array02 +"</h1>");
document.write("<h1>Tamanho 5: "+Array02.length +"</h1>");
document.write("<h1>Array: "+Array02[4] +"</h1>");

Código fonte JavaScript

var Array02 = new Array("textos",456,45.5);

document.write("<h1>Array: "+Array02 +"</h1>");

document.write("<h1>Array: "+Array02.length +"</h1>");

document.write("<h1>Array: "+Array02[0].length +"</h1>");

var verUltimo = Array02[Array02.length - 1];

document.write("<h1>Ultimo: "+verUltimo +"</h1>");

Array02.length = 2;
document.write("<h1>Array: "+Array02 +"</h1>");
document.write("<h1>Tamanho: "+Array02.length +"</h1>");

Array02.length = 5;
document.write("<h1>Array: "+Array02 +"</h1>");
document.write("<h1>Tamanho 5: "+Array02.length +"</h1>");
document.write("<h1>Array: "+Array02[4] +"</h1>");
