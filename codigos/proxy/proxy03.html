Configurar proxy no Windows 10


Como configurar manualmente seu proxy no windows 10.

1º Passo: No campo de busca escreva a palavra proxy;
2° Passo: Selecione o item: Configurações de proxy de rede;
3° Passo: Abrirá a tela: Configuração de proxy automática;
4º Passo: Desça até a opção chamada: Configuração proxy manual;
5° Passo: Ative está opção;
6° Passo: Acesse o link abaixo:
Clique aqui.
7º Passo: Selecione um dos proxys;
8º Passo: No campo endereço novamente no Configuração proxy manual insira o proxy;
9º Passo: No campo porta, insira os ultimos números que aparecem do proxy escolhido;
10º Passo: Desabilite e habilite o proxy manual novamente;
11° Passo: Utilizando o link abaixo verifique de onde é o proxy qye está utilizando.
